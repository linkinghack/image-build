👋 欢迎使用DevSpace! 
🔥 这是一个开发环境容器。您可以使用VSCode for WEB 随时随地在浏览器中开发。
🧩 环境内置丰富的开发环境插件，支持您对云原生微服务系统进行在线开发和调试。
📝 在使用过程中有任何问题，欢迎您与我们取得联系，我们倾听您的意见并及时响应您的需求。

🔍 您可以通过调出命令面板来搜索更多功能。(Cmd/Ctrl + Shift + P or F1)
🛠️ 您可以直接使用`kubectl`, `helm`等命令行工具直接访问当前EKC集群，也可以使用`docker`命令在本地构建镜像或测试容器应用。
---------------
☕️ 当前开发环境类型为: Java+Maven+Gradle
💡当前环境中安装了多个JDK版本,您可以使用命令 `sdk list java`列出当前环境已经安装的Java版本列表,
  使用命令`sdk use java <version>`来切换JDK版本: 例如`sdk use java 17.0.0.0-tem`
